Watchd é um projeto em React + Vite para descobrir séries, explorar listas da comunidade e visualizar títulos populares usando a API do TVMaze.

## Funcionalidades

- Busca de séries por título
- Sugestões aleatórias de séries
- Listas da comunidade com prévia de pôsteres
- Telas de login e cadastro com imagens de séries populares
- Layout responsivo estilizado com Tailwind CSS

## Tecnologias

- React
- Vite
- Tailwind CSS
- Lucide React
- API do TVMaze

## Como Executar

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Gere a versão de produção:

```bash
npm run build
```
